.. _development:

*****************
Development Tools
*****************

本章所述的模块用来协助编写软件。例如，\ :mod:`pydoc` 模块以一个模块为参数，能根据其中的内容生成文档。\ :mod:`doctest` 和 :mod:`unittest` 模块包含编写单元测试的框架，可以自动执行代码并验证其转出的正确性。\ :program:`2to3` 可以把 Python 2.x 的源代码转换成有效的 Python 3.x 代码。

本章所述的模块目录如下:
